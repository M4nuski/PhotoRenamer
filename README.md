Automatic Photo Renamer (Dater2)
----------
Scan les fichiers sélectionnés et renome automatiquement selon la date de prise de vue du fichier.

* Possibilitée de garder le nom original et seulement préfixer la date
* Option d'ajout de suffix
* Renome aussi les .nef (raw nikon) trouvé avec les .jpg


----------
Scan selected files and rename them with date/timestamp of exif data

* "Prefix" will add date/time to original filename
* "Suffix" appends text to each filename
* .nef renaming if nikon raw image is found with jpg preview.
